# Wails 零基础到精通学习计划

> Wails 是一个使用 Go 语言编写后端、Web 技术构建前端的跨平台桌面应用开发框架。
> 与 Electron 不同，Wails 使用系统原生 WebView，生成的二进制文件体积小（比 Electron 小约 90%）、性能高。

---

## 第一阶段：基础准备（1-2 周）

### 1.1 Go 语言基础

> 如果已有 Go 基础可跳过此节。

- [x] Go 基本语法：变量、常量、数据类型、控制流
- [x] 函数、方法、接口
- [x] 结构体（struct）与方法绑定
- [x] 错误处理机制
- [x] Goroutine 与 Channel 基础
- [x] Go Modules 依赖管理
- [x] 常用标准库：`fmt`、`os`、`net/http`、`encoding/json`、`context`

### 1.2 前端基础

> 至少掌握一种前端框架，推荐 Vue 或 React。

- [x] HTML / CSS / JavaScript 基础
- [x] TypeScript 基础（Wails 自动生成 TS 类型定义）
- [x] 掌握至少一个前端框架：
  - **Vue 3**（推荐，上手快）
  - React
  - Svelte
- [x] Node.js 与 npm/pnpm 包管理基础
- [x] Vite 构建工具基础概念

### 1.3 开发环境搭建

- [x] 安装 Go（1.21+）
- [x] 安装 Node.js（18+）
- [x] 安装 IDE：VS Code 或 GoLand
- [x] 安装 Wails CLI：
  ```bash
  go install github.com/wailsapp/wails/v2/cmd/wails@latest
  ```
- [x] 运行环境检查：
  ```bash
  wails doctor
  ```
- [x] 各平台依赖安装：
  - **Windows**：需要 WebView2 Runtime（Win10/11 通常已内置）
  - **macOS**：需要 Xcode Command Line Tools
  - **Linux**：需要 `gtk3`、`webkit2gtk` 等依赖

## 第二阶段：入门实战（2-3 周）

### 2.1 第一个 Wails 应用

- [ ] 使用 CLI 创建项目：
  ```bash
  wails init -n myapp -t vue-ts
  ```
- [ ] 理解项目目录结构：
  ```
  myapp/
  ├── build/              # 构建相关文件（图标、manifest 等）
  ├── frontend/           # 前端代码目录
  │   ├── src/
  │   ├── index.html
  │   ├── package.json
  │   └── vite.config.ts
  ├── app.go              # 应用核心逻辑
  ├── main.go             # 应用入口
  ├── wails.json          # Wails 项目配置
  └── go.mod
  ```
- [ ] 启动开发模式：
  ```bash
  wails dev
  ```
- [ ] 理解热重载机制（前端 + Go 后端均支持）

### 2.2 核心概念：Go 与 JS 的桥接（Bindings）

- [ ] 理解 **绑定（Binding）** 机制：Go 方法暴露给前端调用
- [ ] 编写可绑定的 Go 结构体方法：
  ```go
  type App struct {
      ctx context.Context
  }

  func (a *App) Greet(name string) string {
      return fmt.Sprintf("Hello %s!", name)
  }
  ```
- [ ] 在 `main.go` 中注册绑定：
  ```go
  err := wails.Run(&options.App{
      Title:  "My App",
      Width:  1024,
      Height: 768,
      Bind: []interface{}{
          app,
      },
  })
  ```
- [ ] 前端调用 Go 方法：
  ```typescript
  import { Greet } from '../wailsjs/go/main/App'

  const result = await Greet("World")
  ```
- [ ] 理解自动生成的 TypeScript 类型定义（`wailsjs/` 目录）
- [ ] 数据类型映射：Go struct ↔ TypeScript interface

### 2.3 应用生命周期

- [ ] `OnStartup(ctx context.Context)` — 应用启动时调用
- [ ] `OnDomReady(ctx context.Context)` — DOM 准备就绪
- [ ] `OnShutdown(ctx context.Context)` — 应用关闭时调用
- [ ] `OnBeforeClose(ctx context.Context) bool` — 关闭前拦截
- [ ] 理解 `context.Context` 在 Wails 中的作用

### 2.4 项目配置文件 `wails.json`

- [ ] 核心配置项：
  - `name` — 项目名称
  - `frontend:install` — 前端依赖安装命令
  - `frontend:build` — 前端构建命令
  - `frontend:dev:serverUrl` — 开发服务器 URL
  - `wailsjsdir` — 自动生成 JS 模块的目录
  - `outputfilename` — 输出二进制文件名
- [ ] 构建钩子配置（`preBuildHooks` / `postBuildHooks`）

## 第三阶段：核心功能深入（3-4 周）

### 3.1 事件系统（Events）

- [ ] Go 端发送事件：
  ```go
  runtime.EventsEmit(ctx, "eventName", data)
  ```
- [ ] Go 端监听事件：
  ```go
  runtime.EventsOn(ctx, "eventName", func(args ...interface{}) {
      // 处理事件
  })
  ```
- [ ] 前端发送事件：
  ```typescript
  import { EventsEmit } from '../wailsjs/runtime/runtime'
  EventsEmit("eventName", data)
  ```
- [ ] 前端监听事件：
  ```typescript
  import { EventsOn } from '../wailsjs/runtime/runtime'
  EventsOn("eventName", (data) => {
      // 处理事件
  })
  ```
- [ ] 一次性事件监听：`EventsOnce`
- [ ] 多次事件监听：`EventsOnMultiple`
- [ ] 取消事件监听：`EventsOff`

### 3.2 窗口管理

- [ ] 窗口属性设置：
  - 大小（`Width`/`Height`）、最小/最大尺寸
  - 标题（`Title`）
  - 全屏（`Fullscreen`）
  - 无边框模式（`Frameless`）
  - 窗口居中 / 置顶
- [ ] 运行时窗口 API：
  ```go
  runtime.WindowSetSize(ctx, 800, 600)
  runtime.WindowCenter(ctx)
  runtime.WindowSetTitle(ctx, "New Title")
  runtime.WindowMinimise(ctx)
  runtime.WindowMaximise(ctx)
  runtime.WindowToggleMaximise(ctx)
  runtime.WindowFullscreen(ctx)
  ```
- [ ] 无边框窗口的拖拽实现（`--wails-draggable: drag`）
- [ ] 半透明与毛玻璃效果

### 3.3 原生对话框（Dialogs）

- [ ] 消息对话框：
  ```go
  runtime.MessageDialog(ctx, runtime.MessageDialogOptions{
      Type:    runtime.InfoDialog,
      Title:   "提示",
      Message: "操作成功！",
  })
  ```
- [ ] 文件打开对话框：
  ```go
  selection, err := runtime.OpenFileDialog(ctx, runtime.OpenDialogOptions{
      Title: "选择文件",
      Filters: []runtime.FileFilter{
          {DisplayName: "文本文件", Pattern: "*.txt;*.md"},
      },
  })
  ```
- [ ] 多文件选择对话框：`OpenMultipleFilesDialog`
- [ ] 文件保存对话框：`SaveFileDialog`
- [ ] 目录选择对话框：`OpenDirectoryDialog`
- [ ] 各类对话框类型：`InfoDialog`、`WarningDialog`、`ErrorDialog`、`QuestionDialog`

### 3.4 原生菜单（Menus）

- [ ] 创建应用菜单：
  ```go
  menu := menu.NewMenu()
  fileMenu := menu.AddSubmenu("文件")
  fileMenu.AddText("打开", keys.CmdOrCtrl("o"), openFile)
  fileMenu.AddText("保存", keys.CmdOrCtrl("s"), saveFile)
  fileMenu.AddSeparator()
  fileMenu.AddText("退出", keys.CmdOrCtrl("q"), quit)
  ```
- [ ] 菜单项类型：
  - 文本菜单项（`AddText`）
  - 复选框菜单项（`AddCheckbox`）
  - 单选按钮组（`AddRadio`）
  - 子菜单（`AddSubmenu`）
  - 分隔符（`AddSeparator`）
- [ ] 快捷键绑定
- [ ] 动态更新菜单（运行时）
- [ ] 右键上下文菜单

### 3.5 剪贴板

- [ ] 读取剪贴板：
  ```go
  text, err := runtime.ClipboardGetText(ctx)
  ```
- [ ] 写入剪贴板：
  ```go
  runtime.ClipboardSetText(ctx, "要复制的内容")
  ```

## 第七阶段：Wails v3 新特性（2-3 周）

> Wails v3 目前处于 Alpha 阶段，API 已趋于稳定。

### 7.1 v3 核心变化

- [ ] 从声明式 API 到过程式 API 的转变：
  ```go
  // v2 - 声明式
  wails.Run(&options.App{
      Title: "My App",
      Bind:  []interface{}{&GreetService{}},
  })

  // v3 - 过程式
  app := application.New(application.Options{
      Name: "My App",
      Services: []application.Service{
          application.NewService(&GreetService{}),
      },
  })
  app.Run()
  ```
- [ ] **Services** 概念：取代 v2 的 Bind，更好的代码组织
- [ ] 新的构建系统：使用 Taskfile 编排，更灵活透明
- [ ] 静态分析绑定生成（取代反射）

### 7.2 多窗口支持

- [ ] 创建和管理多个窗口：
  ```go
  window1 := app.Window.NewWithOptions(application.WebviewWindowOptions{
      Title:  "主窗口",
      Width:  1024,
      Height: 768,
  })

  window2 := app.Window.NewWithOptions(application.WebviewWindowOptions{
      Title:  "设置",
      Width:  600,
      Height: 400,
  })
  ```
- [ ] 窗口间通信
- [ ] 窗口生命周期管理

### 7.3 系统托盘（System Tray）

- [ ] 创建系统托盘图标：
  ```go
  tray := app.SystemTray.New()
  tray.SetIcon(iconBytes)
  tray.SetLabel("My App")
  ```
- [ ] 托盘右键菜单：
  ```go
  trayMenu := app.NewMenu()
  trayMenu.Add("显示窗口").OnClick(showWindow)
  trayMenu.Add("退出").OnClick(quit)
  tray.SetMenu(trayMenu)
  ```
- [ ] 托盘点击事件处理

### 7.4 增强的事件系统

- [ ] 类型化事件（Typed Events）
- [ ] 系统事件监听（应用激活、失活、主题变更等）
- [ ] 更简洁的事件 API

### 7.5 v2 到 v3 迁移

- [ ] 应用初始化方式变更
- [ ] 绑定到 Services 的迁移
- [ ] 窗口管理 API 变更

## 学习建议

1. **边学边练**：每学一个知识点，立即在项目中实践
2. **阅读源码**：Wails 的 [examples 目录](https://github.com/wailsapp/wails/tree/master/examples) 是最好的学习材料
3. **前后端均衡**：不要只关注 Go 后端或前端，两者结合才是 Wails 的精髓
4. **关注 v3 动态**：v3 已趋于稳定，新项目建议评估是否直接使用 v3
5. **参与社区**：在 Discord 和 GitHub Discussions 中提问和分享
