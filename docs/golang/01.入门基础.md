# Go 语言入门基础

## 1. 环境搭建与初识 Go

### 1.1 Go 语言简介

Go（又称 Golang）是 Google 于 2009 年发布的开源编程语言，由 Robert Griesemer、Rob Pike 和 Ken Thompson 三位大师设计。

**核心设计哲学：**

- **简洁**：语法精简，关键字仅 25 个
- **高效**：编译速度快，运行性能接近 C
- **并发**：原生支持并发编程（Goroutine + Channel）
- **工程化**：自带格式化、测试、文档等工具链

**Go 的典型应用场景：**

- 云原生基础设施（Docker、Kubernetes）
- 微服务后端开发
- 网络编程与中间件
- 命令行工具（CLI）
- 分布式系统

### 1.2 安装 Go 开发环境

从 [Go 官网](https://go.dev/dl/) 下载对应系统的安装包，安装完成后验证：

```bash
go version
# 输出: go version go1.22.0 windows/amd64
```

查看 Go 环境配置：

```bash
go env
```

**重要环境变量：**

| 变量 | 说明 |
|------|------|
| `GOROOT` | Go 安装目录 |
| `GOPATH` | 工作区目录（默认 `~/go`） |
| `GOPROXY` | 模块代理地址 |
| `GOBIN` | 可执行文件安装目录 |

**配置国内代理（加速依赖下载）：**

```bash
go env -w GOPROXY=https://goproxy.cn,direct
```

### 1.3 第一个 Go 程序

创建项目目录并初始化模块：

```bash
mkdir hello && cd hello
go mod init hello
```

创建 `main.go`：

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

运行程序：

```bash
go run main.go
# 输出: Hello, World!
```

编译为可执行文件：

```bash
go build -o hello main.go
./hello
# 输出: Hello, World!
```

**代码解析：**

- `package main`：声明包名，`main` 包是程序入口包
- `import "fmt"`：导入标准库的 `fmt` 包，用于格式化输入输出
- `func main()`：程序入口函数，无参数无返回值

### 1.4 Go 常用命令

```bash
go run main.go      # 编译并直接运行
go build             # 编译生成可执行文件
go install           # 编译并安装到 $GOBIN
go fmt ./...         # 格式化代码
go vet ./...         # 静态分析检查
go test ./...        # 运行测试
go mod init <module> # 初始化模块
go mod tidy          # 整理依赖（添加缺失、移除多余）
go get <package>     # 下载并安装依赖包
go doc fmt.Println   # 查看文档
```

### 1.5 Go Modules 依赖管理

Go Modules 是 Go 官方的依赖管理方案（Go 1.11 引入，Go 1.16 默认启用）。

初始化模块后会生成 `go.mod` 文件：

```
module hello

go 1.22
```

添加第三方依赖示例：

```bash
go get github.com/gin-gonic/gin
```

执行后 `go.mod` 会自动更新，同时生成 `go.sum` 文件记录依赖的哈希校验值。

## 2. 基本语法

### 2.1 变量声明与赋值

Go 是**静态类型**语言，变量必须有明确的类型。

```go
package main

import "fmt"

func main() {
    // 方式一：var 声明（显式指定类型）
    var name string = "Go"
    var age int = 15

    // 方式二：var 声明（类型推断）
    var version = "1.22"

    // 方式三：短变量声明（只能在函数内部使用）
    lang := "Golang"

    // 方式四：批量声明
    var (
        width  int     = 100
        height int     = 200
        pi     float64 = 3.14
    )

    // 零值：未赋值的变量会初始化为类型的零值
    var count int      // 0
    var flag bool      // false
    var msg string     // ""（空字符串）
    var ptr *int       // nil

    fmt.Println(name, age)       // Go 15
    fmt.Println(version)         // 1.22
    fmt.Println(lang)            // Golang
    fmt.Println(width, height)   // 100 200
    fmt.Println(pi)              // 3.14
    fmt.Println(count, flag)     // 0 false
    fmt.Println(msg, ptr)        //  <nil>
}
```

**注意事项：**

- Go 中声明的变量**必须使用**，否则编译报错
- `:=` 只能用在函数内部，不能用于包级别变量
- 多重赋值：`a, b = b, a`（交换变量值）

### 2.2 基本数据类型

```go
package main

import (
    "fmt"
    "unsafe"
)

func main() {
    // 整型
    var i8 int8 = 127           // -128 ~ 127
    var u8 uint8 = 255          // 0 ~ 255
    var i int = 42              // 平台相关：32位或64位
    fmt.Println(i8, u8, i)      // 127 255 42

    // 浮点型
    var f32 float32 = 3.14
    var f64 float64 = 3.141592653589793
    fmt.Println(f32)            // 3.14
    fmt.Println(f64)            // 3.141592653589793

    // 布尔型
    var ok bool = true
    fmt.Println(ok)             // true

    // 字符串（不可变的字节序列）
    var s string = "Hello, 世界"
    fmt.Println(s)              // Hello, 世界
    fmt.Println(len(s))         // 13（字节数，中文占3字节）

    // byte 和 rune
    var b byte = 'A'            // byte 是 uint8 的别名
    var r rune = '中'           // rune 是 int32 的别名，表示 Unicode 码点
    fmt.Println(b, r)           // 65 20013

    // 查看类型和大小
    fmt.Printf("int 大小: %d 字节\n", unsafe.Sizeof(i))
    fmt.Printf("string 类型: %T\n", s)
}
```

**整型一览：**

| 类型 | 大小 | 范围 |
|------|------|------|
| `int8` / `uint8` | 1 字节 | -128~127 / 0~255 |
| `int16` / `uint16` | 2 字节 | -32768~32767 / 0~65535 |
| `int32` / `uint32` | 4 字节 | -2^31~2^31-1 / 0~2^32-1 |
| `int64` / `uint64` | 8 字节 | -2^63~2^63-1 / 0~2^64-1 |
| `int` / `uint` | 平台相关 | 32 位或 64 位 |

### 2.3 常量与枚举

```go
package main

import "fmt"

// 常量声明
const PI = 3.14159
const AppName string = "MyApp"

// 批量声明
const (
    StatusOK    = 200
    StatusError = 500
)

// iota：常量生成器，从 0 开始自增
const (
    Sunday    = iota // 0
    Monday           // 1
    Tuesday          // 2
    Wednesday        // 3
    Thursday         // 4
    Friday           // 5
    Saturday         // 6
)

// iota 实现位掩码
const (
    ReadPermission   = 1 << iota // 1  (001)
    WritePermission              // 2  (010)
    ExecutePermission            // 4  (100)
)

// iota 实现存储单位
const (
    _  = iota             // 忽略第一个值 0
    KB = 1 << (10 * iota) // 1 << 10 = 1024
    MB                    // 1 << 20 = 1048576
    GB                    // 1 << 30 = 1073741824
    TB                    // 1 << 40
)

func main() {
    fmt.Println("PI:", PI)                     // PI: 3.14159
    fmt.Println("AppName:", AppName)           // AppName: MyApp
    fmt.Println("StatusOK:", StatusOK)         // StatusOK: 200

    fmt.Println("Sunday:", Sunday)             // Sunday: 0
    fmt.Println("Monday:", Monday)             // Monday: 1
    fmt.Println("Saturday:", Saturday)         // Saturday: 6

    fmt.Println("Read:", ReadPermission)       // Read: 1
    fmt.Println("Write:", WritePermission)     // Write: 2
    fmt.Println("Execute:", ExecutePermission) // Execute: 4

    fmt.Println("KB:", KB)                     // KB: 1024
    fmt.Println("MB:", MB)                     // MB: 1048576
    fmt.Println("GB:", GB)                     // GB: 1073741824
}
```

### 2.4 类型转换

Go 不支持隐式类型转换，所有类型转换必须**显式**进行。

```go
package main

import (
    "fmt"
    "strconv"
)

func main() {
    // 数值类型之间的转换
    var i int = 42
    var f float64 = float64(i)
    var u uint8 = uint8(f)
    fmt.Println(i, f, u)  // 42 42 42

    // 整数 → 字符串
    s1 := strconv.Itoa(42)
    fmt.Println(s1)        // "42"

    // 字符串 → 整数
    n, err := strconv.Atoi("100")
    if err != nil {
        fmt.Println("转换失败:", err)
    } else {
        fmt.Println(n)     // 100
    }

    // 字符串 → 浮点数
    f2, _ := strconv.ParseFloat("3.14", 64)
    fmt.Println(f2)        // 3.14

    // 浮点数 → 字符串
    s2 := strconv.FormatFloat(3.14, 'f', 2, 64)
    fmt.Println(s2)        // 3.14

    // 布尔值 → 字符串
    s3 := strconv.FormatBool(true)
    fmt.Println(s3)        // true

    // 字符串 → 布尔值
    b, _ := strconv.ParseBool("true")
    fmt.Println(b)         // true

    // 字符串与字节切片互转
    str := "Hello"
    bytes := []byte(str)
    fmt.Println(bytes)     // [72 101 108 108 111]
    str2 := string(bytes)
    fmt.Println(str2)      // Hello
}
```

### 2.5 运算符

```go
package main

import "fmt"

func main() {
    // 算术运算符
    fmt.Println(10 + 3)   // 13
    fmt.Println(10 - 3)   // 7
    fmt.Println(10 * 3)   // 30
    fmt.Println(10 / 3)   // 3（整数除法，截断小数）
    fmt.Println(10 % 3)   // 1（取余）
    fmt.Println(10.0 / 3) // 3.3333333333333335

    // 关系运算符
    fmt.Println(3 == 3) // true
    fmt.Println(3 != 4) // true
    fmt.Println(3 > 4)  // false
    fmt.Println(3 < 4)  // true
    fmt.Println(3 >= 3) // true
    fmt.Println(3 <= 2) // false

    // 逻辑运算符
    fmt.Println(true && false) // false
    fmt.Println(true || false) // true
    fmt.Println(!true)         // false

    // 位运算符
    a, b := 0b1010, 0b1100 // 10, 12
    fmt.Printf("AND:  %04b\n", a&b)  // AND:  1000
    fmt.Printf("OR:   %04b\n", a|b)  // OR:   1110
    fmt.Printf("XOR:  %04b\n", a^b)  // XOR:  0110
    fmt.Printf("左移: %04b\n", a<<1) // 左移: 10100
    fmt.Printf("右移: %04b\n", a>>1) // 右移: 0101

    // 自增自减（只能作为语句，不能作为表达式）
    x := 5
    x++
    fmt.Println(x) // 6
    x--
    fmt.Println(x) // 5
    // y := x++  // 编译错误！Go 不允许这样写
}
```

### 2.6 格式化输入输出

```go
package main

import "fmt"

type User struct {
    Name string
    Age  int
}

func main() {
    name := "Go"
    age := 15
    pi := 3.14159
    user := User{Name: "Alice", Age: 25}

    // 通用占位符
    fmt.Printf("值:   %v\n", user)   // 值:   {Alice 25}
    fmt.Printf("详细: %+v\n", user)  // 详细: {Name:Alice Age:25}
    fmt.Printf("语法: %#v\n", user)  // 语法: main.User{Name:"Alice", Age:25}
    fmt.Printf("类型: %T\n", user)   // 类型: main.User

    // 整数占位符
    fmt.Printf("十进制:   %d\n", age)     // 十进制:   15
    fmt.Printf("二进制:   %b\n", age)     // 二进制:   1111
    fmt.Printf("八进制:   %o\n", age)     // 八进制:   17
    fmt.Printf("十六进制: %x\n", age)     // 十六进制: f
    fmt.Printf("字符:     %c\n", 65)      // 字符:     A

    // 浮点数占位符
    fmt.Printf("默认:     %f\n", pi)      // 默认:     3.141590
    fmt.Printf("两位小数: %.2f\n", pi)    // 两位小数: 3.14
    fmt.Printf("科学计数: %e\n", pi)      // 科学计数: 3.141590e+00

    // 字符串占位符
    fmt.Printf("字符串:   %s\n", name)    // 字符串:   Go
    fmt.Printf("带引号:   %q\n", name)    // 带引号:   "Go"

    // 宽度与对齐
    fmt.Printf("|%-10s|%10s|\n", "左对齐", "右对齐")
    // |左对齐       |       右对齐|
    fmt.Printf("|%-10d|%010d|\n", 42, 42)
    // |42        |0000000042|

    // Sprintf：格式化为字符串（不输出）
    result := fmt.Sprintf("%s 已经 %d 岁了", name, age)
    fmt.Println(result) // Go 已经 15 岁了

    // Println vs Print vs Printf
    fmt.Println("A", "B", "C") // A B C（自动空格 + 换行）
    fmt.Print("A", "B", "C")   // ABC（无空格、无换行）
    fmt.Println()
}
```

**常用格式化占位符速查：**

| 占位符 | 说明 | 示例 |
|--------|------|------|
| `%v` | 默认格式 | `{Alice 25}` |
| `%+v` | 带字段名 | `{Name:Alice Age:25}` |
| `%#v` | Go 语法表示 | `main.User{...}` |
| `%T` | 类型 | `main.User` |
| `%d` | 十进制整数 | `42` |
| `%b` | 二进制 | `101010` |
| `%o` | 八进制 | `52` |
| `%x` | 十六进制 | `2a` |
| `%f` | 浮点数 | `3.141590` |
| `%.2f` | 指定精度 | `3.14` |
| `%s` | 字符串 | `hello` |
| `%q` | 带引号字符串 | `"hello"` |
| `%p` | 指针地址 | `0xc0000b4008` |
| `%%` | 百分号字面量 | `%` |

## 3. 流程控制

### 3.1 条件语句 if

```go
package main

import "fmt"

func main() {
    age := 20

    // 基本 if-else
    if age >= 18 {
        fmt.Println("成年人")
    } else {
        fmt.Println("未成年人")
    }
    // 输出: 成年人

    // if-else if-else
    score := 85
    if score >= 90 {
        fmt.Println("优秀")
    } else if score >= 80 {
        fmt.Println("良好")
    } else if score >= 60 {
        fmt.Println("及格")
    } else {
        fmt.Println("不及格")
    }
    // 输出: 良好

    // if 带初始化语句（变量作用域仅限 if 块内）
    if num := 10 * 2; num > 15 {
        fmt.Println("num 大于 15, num =", num)
    } else {
        fmt.Println("num 不大于 15, num =", num)
    }
    // 输出: num 大于 15, num = 20
    // fmt.Println(num) // 编译错误：num 在 if 块外不可见
}
```

### 3.2 循环语句 for

Go **只有 `for`** 一种循环语句，没有 `while` 和 `do-while`。

```go
package main

import "fmt"

func main() {
    // 标准 for 循环
    for i := 0; i < 5; i++ {
        fmt.Print(i, " ")
    }
    fmt.Println()
    // 输出: 0 1 2 3 4

    // 类似 while 的写法
    count := 0
    for count < 3 {
        fmt.Print(count, " ")
        count++
    }
    fmt.Println()
    // 输出: 0 1 2

    // 无限循环（配合 break 退出）
    sum := 0
    for {
        sum++
        if sum >= 5 {
            break
        }
    }
    fmt.Println("sum:", sum)
    // 输出: sum: 5

    // for-range 遍历字符串
    str := "Hi你好"
    for i, ch := range str {
        fmt.Printf("索引:%d 字符:%c Unicode:%d\n", i, ch, ch)
    }
    // 输出:
    // 索引:0 字符:H Unicode:72
    // 索引:1 字符:i Unicode:105
    // 索引:2 字符:你 Unicode:20320
    // 索引:5 字符:好 Unicode:22909

    // for-range 遍历切片
    fruits := []string{"苹果", "香蕉", "橙子"}
    for index, value := range fruits {
        fmt.Printf("索引:%d 值:%s\n", index, value)
    }
    // 输出:
    // 索引:0 值:苹果
    // 索引:1 值:香蕉
    // 索引:2 值:橙子

    // 只需要索引时，省略第二个变量
    for i := range fruits {
        fmt.Print(i, " ")
    }
    fmt.Println()
    // 输出: 0 1 2

    // 不需要索引时，用 _ 忽略
    for _, fruit := range fruits {
        fmt.Print(fruit, " ")
    }
    fmt.Println()
    // 输出: 苹果 香蕉 橙子
}
```

### 3.3 break、continue 与标签

```go
package main

import "fmt"

func main() {
    // continue：跳过本次循环
    fmt.Print("奇数: ")
    for i := 0; i < 10; i++ {
        if i%2 == 0 {
            continue // 跳过偶数
        }
        fmt.Print(i, " ")
    }
    fmt.Println()
    // 输出: 奇数: 1 3 5 7 9

    // break：退出循环
    fmt.Print("找到第一个大于5的偶数: ")
    for i := 0; i < 20; i++ {
        if i > 5 && i%2 == 0 {
            fmt.Println(i)
            break
        }
    }
    // 输出: 找到第一个大于5的偶数: 6

    // 标签（Label）：跳出多层嵌套循环
    fmt.Println("标签示例:")
outer:
    for i := 0; i < 3; i++ {
        for j := 0; j < 3; j++ {
            if i == 1 && j == 1 {
                fmt.Println("  在 i=1, j=1 处跳出外层循环")
                break outer // 直接跳出外层循环
            }
            fmt.Printf("  i=%d, j=%d\n", i, j)
        }
    }
}
```

### 3.4 switch 语句

```go
package main

import "fmt"

func main() {
    // 基本 switch
    day := 3
    switch day {
    case 1:
        fmt.Println("星期一")
    case 2:
        fmt.Println("星期二")
    case 3:
        fmt.Println("星期三")
    case 4:
        fmt.Println("星期四")
    case 5:
        fmt.Println("星期五")
    default:
        fmt.Println("周末")
    }
    // 输出: 星期三

    // 多值匹配
    month := 6
    switch month {
    case 3, 4, 5:
        fmt.Println("春天")
    case 6, 7, 8:
        fmt.Println("夏天")
    case 9, 10, 11:
        fmt.Println("秋天")
    case 12, 1, 2:
        fmt.Println("冬天")
    }
    // 输出: 夏天

    // switch 带初始化语句
    switch num := 15; {
    case num < 0:
        fmt.Println("负数")
    case num == 0:
        fmt.Println("零")
    case num > 0 && num <= 10:
        fmt.Println("小正数")
    default:
        fmt.Println("大正数")
    }
    // 输出: 大正数

    // 无条件 switch（替代 if-else 链）
    score := 85
    switch {
    case score >= 90:
        fmt.Println("A")
    case score >= 80:
        fmt.Println("B")
    case score >= 70:
        fmt.Println("C")
    case score >= 60:
        fmt.Println("D")
    default:
        fmt.Println("F")
    }
    // 输出: B

    // fallthrough：强制执行下一个 case（不判断条件）
    n := 1
    switch n {
    case 1:
        fmt.Print("一 ")
        fallthrough
    case 2:
        fmt.Print("二 ")
        fallthrough
    case 3:
        fmt.Print("三")
    }
    fmt.Println()
    // 输出: 一 二 三

    // 类型 switch
    var value interface{} = "hello"
    switch v := value.(type) {
    case int:
        fmt.Printf("整数: %d\n", v)
    case string:
        fmt.Printf("字符串: %s\n", v)
    case bool:
        fmt.Printf("布尔值: %t\n", v)
    default:
        fmt.Printf("未知类型: %T\n", v)
    }
    // 输出: 字符串: hello
}
```

**Go 的 switch 与其他语言的区别：**

- 每个 `case` 默认自带 `break`，不会穿透到下一个 `case`
- 需要穿透时使用 `fallthrough` 关键字
- `case` 后可以跟多个值，用逗号分隔
- 可以没有判断条件（无条件 `switch`），替代 `if-else` 链

## 4. 综合练习

### 练习 1：猜数字游戏

```go
package main

import (
    "fmt"
    "math/rand"
)

func main() {
    // 生成 1-100 的随机数
    target := rand.IntN(100) + 1
    fmt.Println("猜数字游戏！范围 1-100")
    fmt.Println("（提示：答案是", target, "）") // 调试用

    for attempts := 1; ; attempts++ {
        var guess int
        fmt.Print("请输入你的猜测: ")
        fmt.Scan(&guess)

        switch {
        case guess < target:
            fmt.Println("太小了！")
        case guess > target:
            fmt.Println("太大了！")
        default:
            fmt.Printf("恭喜！你猜对了！用了 %d 次\n", attempts)
            return
        }
    }
}
```
