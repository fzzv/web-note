# Go 语言学习计划：从零基础到精通

## 第一阶段：入门基础（1-2 周）

### 1. 环境搭建与初识 Go

- Go 语言的历史与设计哲学（简洁、高效、并发）
- 安装 Go 开发环境（Go SDK、GOPATH、Go Modules）
- 配置 IDE / 编辑器（VS Code + Go 插件 / GoLand）
- 第一个 Go 程序：`Hello, World!`
- `go run`、`go build`、`go install` 基本命令
- Go Modules 依赖管理（`go mod init`、`go mod tidy`）

### 2. 基本语法

- 变量声明与赋值（`var`、`:=`、多变量声明）
- 基本数据类型（`int`、`float64`、`string`、`bool`、`byte`、`rune`）
- 常量与枚举（`const`、`iota`）
- 类型转换与类型别名
- 运算符（算术、关系、逻辑、位运算）
- 输入输出（`fmt` 包：`Println`、`Printf`、`Scanf`）

### 3. 流程控制

- 条件语句（`if`、`if-else`、`if` 初始化语句）
- 循环语句（`for`、`for-range`，Go 没有 `while`）
- `switch` 语句（无需 `break`、多值匹配、类型 `switch`）
- `goto`、`break`、`continue` 与标签

---

## 第二阶段：核心语法（2-3 周）

### 4. 函数

- 函数定义与调用
- 多返回值
- 命名返回值
- 可变参数（`...`）
- 匿名函数与闭包
- 函数作为值和参数传递
- `defer` 语句（执行顺序、资源释放）
- `panic` 与 `recover`（错误恢复机制）

### 5. 复合数据类型

- 数组（定长、值类型）
- 切片（`slice`）：创建、追加、截取、扩容机制
- 映射（`map`）：创建、增删改查、遍历、并发安全问题
- 字符串：底层结构、常用操作、`strings` 包、`strconv` 包
- 指针：`&` 取地址、`*` 解引用、指针与值传递的区别

### 6. 结构体与方法

- 结构体定义与初始化
- 匿名字段与嵌套结构体
- 结构体标签（`tag`）
- 方法定义（值接收者 vs 指针接收者）
- 方法集的概念
- 构造函数模式（Go 没有构造函数，使用工厂函数）

---

## 第三阶段：面向接口编程（2 周）

### 7. 接口

- 接口的定义与实现（隐式实现）
- 空接口 `interface{}` 与 `any`
- 类型断言（`value.(Type)`）
- 类型选择（`type switch`）
- 接口组合（嵌入接口）
- 常用标准库接口：`io.Reader`、`io.Writer`、`fmt.Stringer`、`error`
- 接口设计原则：面向接口编程、小接口优于大接口

### 8. 错误处理

- `error` 接口
- 自定义错误类型
- 错误包装与解包（`fmt.Errorf` + `%w`、`errors.Is`、`errors.As`）
- 错误处理最佳实践（不要忽略错误、错误应只处理一次）
- `panic` / `recover` 的适用场景
- 哨兵错误（Sentinel Errors）vs 自定义错误类型

---

## 第四阶段：并发编程（3-4 周）

### 9. Goroutine

- Goroutine 的概念与创建（`go` 关键字）
- Goroutine 的调度模型（GMP 模型概述）
- `sync.WaitGroup`：等待一组 goroutine 完成
- Goroutine 泄漏与预防

### 10. Channel

- Channel 的创建与使用（无缓冲 vs 有缓冲）
- Channel 的方向（只读、只写）
- `select` 多路复用
- Channel 的关闭与 `range` 遍历
- 常见并发模式：
  - 生产者-消费者
  - 扇入（Fan-in）/ 扇出（Fan-out）
  - Pipeline 管道模式
  - 超时控制与取消
  - 工作池（Worker Pool）

### 11. 同步原语

- `sync.Mutex` / `sync.RWMutex`（互斥锁 / 读写锁）
- `sync.Once`（单次执行）
- `sync.Map`（并发安全 Map）
- `sync.Pool`（对象池）
- `sync.Cond`（条件变量）
- `atomic` 包（原子操作）

### 12. Context

- `context.Background()` / `context.TODO()`
- `context.WithCancel`（取消信号传播）
- `context.WithTimeout` / `context.WithDeadline`（超时控制）
- `context.WithValue`（请求范围值传递）
- Context 在 HTTP 服务、数据库操作中的应用
- Context 使用规范与最佳实践

---

## 第五阶段：标准库与工程实践（3-4 周）

### 13. 常用标准库

- `os` / `io` / `bufio`：文件与 I/O 操作
- `encoding/json`：JSON 序列化 / 反序列化、结构体标签
- `encoding/xml`、`encoding/csv`
- `net/http`：HTTP 客户端与服务端
- `net/url`：URL 解析
- `time`：时间处理、定时器、格式化（`2006-01-02 15:04:05`）
- `regexp`：正则表达式
- `sort`：排序
- `math` / `math/rand`：数学运算与随机数
- `log` / `log/slog`：日志
- `path` / `filepath`：路径处理
- `flag`：命令行参数解析

### 14. 测试

- 单元测试（`testing` 包、`go test`）
- 表驱动测试（Table-Driven Tests）
- 基准测试（`Benchmark`）
- 示例测试（`Example`）
- 测试覆盖率（`go test -cover`）
- 模糊测试（Fuzz Testing，Go 1.18+）
- 测试辅助：`testify`、`gomock` / `mockgen`
- `httptest` 包：HTTP 处理器测试

### 15. 项目工程化

- 项目目录结构规范（参考 `golang-standards/project-layout`）
- Go Modules 高级用法（`replace`、私有模块、`go work`）
- 代码规范与工具链：
  - `gofmt` / `goimports`（代码格式化）
  - `golangci-lint`（静态分析）
  - `go vet`（代码检查）
- Makefile 编写
- 配置管理（`viper`）
- 日志框架（`zap`、`zerolog`）

---

## 第六阶段：泛型与高级特性（2 周）

### 16. 泛型（Go 1.18+）

- 类型参数（Type Parameters）
- 类型约束（Type Constraints）
- `constraints` 包与自定义约束
- 泛型函数与泛型类型
- 泛型接口
- 泛型的适用场景与限制
- 标准库中的泛型：`slices`、`maps`、`cmp` 包

### 17. 反射（Reflect）

- `reflect.Type` 与 `reflect.Value`
- 结构体字段遍历与标签读取
- 动态调用方法
- 反射的性能代价与使用原则
- 反射在 JSON 解析、ORM 中的应用

### 18. unsafe 与 CGO

- `unsafe.Pointer`：指针类型转换
- `unsafe.Sizeof`、`unsafe.Offsetof`
- CGO 基础：Go 调用 C 代码
- CGO 的使用场景与限制

---

## 第七阶段：Web 开发（3-4 周）

### 19. HTTP 服务开发

- `net/http` 标准库深入
  - `Handler` 接口与 `HandlerFunc`
  - `ServeMux` 路由
  - 中间件模式
- 主流 Web 框架：
  - **Gin**：路由、中间件、参数绑定、验证、分组路由
  - **Echo** / **Fiber**（了解）
- Go 1.22+ 增强路由（`http.NewServeMux` 的路径参数支持）
- RESTful API 设计
- 请求参数校验（`validator`）
- 优雅关闭（Graceful Shutdown）

### 20. 数据库操作

- `database/sql` 标准库
- SQL 驱动（`go-sql-driver/mysql`、`lib/pq`、`mattn/go-sqlite3`）
- 连接池配置与管理
- ORM 框架：**GORM**
  - 模型定义与迁移
  - CRUD 操作
  - 关联关系
  - 事务处理
  - 钩子（Hooks）
- Redis 操作（`go-redis`）
- MongoDB 操作（`mongo-go-driver`）（了解）

### 21. 认证与安全

- JWT 认证（`golang-jwt/jwt`）
- Session 管理
- 密码哈希（`bcrypt`）
- CORS 处理
- HTTPS / TLS 配置
- 常见安全问题与防范（SQL 注入、XSS、CSRF）

### 22. API 文档与规范

- Swagger / OpenAPI（`swaggo/swag`）
- Protocol Buffers 与 gRPC
  - `.proto` 文件定义
  - 代码生成
  - 一元 RPC / 流式 RPC
  - 拦截器（Interceptor）

---

## 第八阶段：微服务与分布式（3-4 周）

### 23. 微服务基础

- 微服务架构概念与设计原则
- 服务注册与发现（Consul、etcd）
- 配置中心（Nacos、Apollo）
- API 网关
- 服务间通信：HTTP vs gRPC
- Go 微服务框架（了解）：
  - **Go-Zero**
  - **Go-Micro**
  - **Kratos**

### 24. 消息队列

- Kafka 客户端（`confluent-kafka-go`、`sarama`）
- RabbitMQ / NATS（了解）
- 消息的生产、消费、确认
- 消息幂等性与可靠性保证

### 25. 分布式系统要素

- 分布式锁（Redis / etcd）
- 分布式事务（Saga、TCC 模式）
- 限流与熔断（`golang.org/x/time/rate`、`hystrix-go`）
- 链路追踪（OpenTelemetry、Jaeger）
- 服务监控（Prometheus + Grafana）

---

## 第九阶段：Go 语言深入（2-3 周）

### 26. 内存管理与垃圾回收

- Go 内存分配模型（栈 vs 堆）
- 逃逸分析（`go build -gcflags="-m"`）
- 垃圾回收机制（三色标记法、写屏障）
- GC 调优（`GOGC`、`debug.SetGCPercent`）
- 内存对齐与结构体布局优化

### 27. 调度器与运行时

- GMP 调度模型详解（G、M、P）
- 抢占式调度（Go 1.14+ 信号抢占）
- 系统调用对调度的影响
- `GOMAXPROCS` 的作用
- `runtime` 包常用函数

### 28. 性能优化

- 性能剖析工具：
  - `pprof`（CPU、内存、goroutine、block）
  - `trace` 工具
  - `benchmark` 测试
- 常见优化技巧：
  - 减少内存分配（对象复用、`sync.Pool`）
  - 字符串拼接优化（`strings.Builder`）
  - 切片预分配容量
  - 避免不必要的类型转换
  - 减少锁竞争
- 编译优化选项（`-ldflags`、内联、逃逸分析）

### 29. 代码生成

- `go generate` 机制
- 常见代码生成工具：
  - `stringer`（枚举字符串生成）
  - `mockgen`（Mock 生成）
  - `protoc`（Protobuf 代码生成）
  - `wire`（依赖注入）
  - `ent`（ORM 代码生成）

### 30. 容器化与部署

- Docker 容器化：

  - Docker Compose 本地开发环境

  - 自动化测试、构建、部署
- 交叉编译（`GOOS`、`GOARCH`）
- 静态编译与最小化部署

### 31. 可观测性

- 结构化日志（`slog`、`zap`）
- 指标监控（Prometheus 客户端）
- 链路追踪（OpenTelemetry SDK）
- 健康检查与就绪探针
- 告警配置

## 第十一阶段：实战项目（持续）

### 推荐练习项目（由易到难）

1. **命令行工具**：文件批量重命名 / TODO 列表管理
2. **Web 爬虫**：并发抓取网页、解析数据、存储到数据库
3. **RESTful API 服务**：用户管理系统（注册、登录、CRUD、JWT 认证）
4. **博客系统**：Gin + GORM + MySQL，包含文章、分类、标签、评论
5. **即时通讯**：WebSocket 实时聊天室
6. **短链接服务**：短链生成、重定向、统计访问量
7. **分布式任务调度系统**：定时任务、任务分发、结果收集
8. **微服务电商系统**：用户服务、商品服务、订单服务、支付服务
