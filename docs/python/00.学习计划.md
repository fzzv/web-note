## 二、进阶篇

### 04. 函数

- 函数定义与调用（`def`、`return`）
- 参数类型：位置参数、默认参数、可变参数（`*args`、`**kwargs`）、仅关键字参数
- 作用域与闭包（LEGB 规则、`global`、`nonlocal`）
- 匿名函数 `lambda`
- 高阶函数（`map`、`filter`、`reduce`、`sorted` 的 `key` 参数）
- 递归函数
- 函数注解与类型提示（Type Hints）
- `functools` 常用工具（`partial`、`lru_cache`、`wraps`）

### 05. 面向对象编程（OOP）

- 类与对象（`class`、`__init__`、`self`）
- 属性与方法（实例属性、类属性、实例方法、类方法 `@classmethod`、静态方法 `@staticmethod`）
- 封装（私有属性 `_`/`__`、`@property`）
- 继承与多态（单继承、多继承、`super()`、MRO）
- 魔术方法 / 双下方法（`__str__`、`__repr__`、`__len__`、`__eq__`、`__lt__`、`__add__`、`__getitem__`、`__iter__` 等）
- 抽象基类（`abc` 模块）
- 数据类（`@dataclass`，Python 3.7+）
- 枚举类（`enum` 模块）

### 06. 模块与包

- 模块的导入方式（`import`、`from...import`、`as` 别名）
- `__name__` 与 `__main__`
- 包的结构（`__init__.py`）
- 相对导入与绝对导入
- 常用标准库概览（`os`、`sys`、`json`、`re`、`datetime`、`pathlib`、`logging`、`collections`、`itertools`）
- 第三方包管理（`pip`、`venv`/`virtualenv`、`requirements.txt`）
- `poetry` / `uv` 等现代包管理工具

### 07. 文件与 IO 操作

- 文件读写（`open`、`read`、`write`、`with` 语句）
- 文件模式（`r`、`w`、`a`、`b`、`+`）
- 逐行读取与大文件处理
- `pathlib` 路径操作（Path 对象、路径拼接、遍历目录）
- JSON / CSV / YAML 文件处理
- `pickle` 序列化
- 编码问题处理（UTF-8、GBK）

### 08. 字符串与正则表达式

- `re` 模块详解（`match`、`search`、`findall`、`sub`、`split`）
- 正则语法（字符类、量词、分组、断言、贪婪/非贪婪）
- 编译正则（`re.compile`）
- 常见正则实战（邮箱、手机号、URL、IP 地址）

## 三、高级篇

### 09. 装饰器

- 闭包回顾
- 函数装饰器（无参/带参）
- `@wraps` 保留函数元信息
- 类装饰器
- 常见应用场景（计时、日志、权限校验、缓存、重试）
- 多层装饰器的执行顺序

### 10. 迭代器与生成器

- 可迭代对象（Iterable）与迭代器（Iterator）协议
- `__iter__` 与 `__next__`
- `yield` 与生成器函数
- `yield from` 委托生成器
- 生成器表达式 vs 列表推导式
- `itertools` 模块（`chain`、`product`、`combinations`、`permutations`、`groupby`、`islice`）
- 惰性求值与内存优化

### 11. 并发与异步编程

- 多线程（`threading` 模块、GIL 的影响）
- 多进程（`multiprocessing` 模块）
- 线程池与进程池（`concurrent.futures`）
- 协程基础（`async`/`await`）
- `asyncio` 事件循环
- 异步 IO 应用场景（网络请求、文件 IO）
- `aiohttp` / `httpx` 异步 HTTP 客户端

### 12. 元编程与高级特性

- 描述符协议（`__get__`、`__set__`、`__delete__`）
- 元类（`type`、`__metaclass__`、`__new__` vs `__init__`）
- `__slots__` 优化内存
- 上下文管理器进阶（`contextlib`）
- 动态属性（`__getattr__`、`__setattr__`、`__getattribute__`）

### 13. 类型系统与代码质量

- 类型提示进阶（`typing` 模块：`Optional`、`Union`、`List`、`Dict`、`TypeVar`、`Generic`、`Protocol`）
- `mypy` 静态类型检查
- 单元测试（`unittest`、`pytest`）
- 代码规范（PEP 8、`black`、`ruff`）
- 文档字符串（docstring 规范）

## 学习建议

1. **循序渐进**：按照章节顺序学习，每个章节的知识点都是后续内容的基础
2. **动手实践**：每学完一个主题，写一个小项目或练习来巩固
3. **阅读源码**：尝试阅读优秀开源项目的代码，学习 Pythonic 的写法
4. **刷题练习**：推荐 LeetCode、牛客网，用 Python 刷算法题提升编程能力
5. **实战项目建议**：
   - 进阶篇完成后：写一个命令行待办事项工具
   - 高级篇完成后：写一个简易 Web 框架或爬虫框架
   - 实战篇完成后：完成一个全栈 Web 应用
